# NLW AI

![Página inicial do site](assets/tela-inicial.jpeg "Página inicial do site")

Projeto criado na imersão NLW AI da Rocketseat

## Como executar

Em seu terminal, execute os comando abaixo

```bash
pnpm install

pnpm run dev
```

## Tecnologias

- React
- Vite
- TypeScript
- TailwindCSS
- Radix UI
- Shadcn/ui
